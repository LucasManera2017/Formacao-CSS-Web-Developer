Nomenclaturas: - grid container: elemento pai que terá a propriedade display: flex especificada. Ele irá "envolver" todos os itens do grid - grid item: Elementos que são filhos diretos do grid container - grid columns: Faixas verticais dos itens do grid - grid row: Faixas horizonatais dos itens do grid - grid gap: Espaços entre as linhas e colunas do grid - grid line: Divisórias que compõem a estrutura da grade. As linhas entre as colunas são chamadas de column lines e as que estão entre as faixas horizontais, são chamadas de row lines; - grid cell: São as células, ou seja, cada "quadrado" da grade;
