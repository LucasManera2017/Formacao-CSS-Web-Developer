flex-shrink Controla o quanto esse item vai diminuir caso seja necessário Se o container não possuir espço suficiente para acomodar os elementos, eless irão diminuir para caber. mesmo que tenha um height ou width atribuido, irá encolher para caber. Se colocar min ou max, irá respeitar. flex-shrink: 1 flex-shrink: 2
